# 贡献指南

感谢您对闲鱼自动收发货工具的兴趣！我们欢迎各种形式的贡献。

## 🚀 如何贡献

### 报告 Bug

如果您发现了 bug，请提交 Issue，并包含以下信息：
- 问题的简要描述
- 复现步骤
- 期望行为
- 实际行为
- 系统环境（操作系统、Python 版本等）
- 相关日志或截图

### 提交功能建议

如果您有新功能的想法，请提交 Issue，并包含：
- 功能描述
- 使用场景
- 可能的实现方案（可选）

### 提交代码

1. Fork 本项目
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request

## 📝 代码规范

- 遵循 PEP 8 Python 代码规范
- 添加必要的注释
- 保持代码简洁清晰
- 提交前测试代码

## 🧪 测试

```bash
# 运行测试
python -m pytest tests/

# 检查代码风格
flake8 .

# 类型检查
mypy .
```

## 📚 文档

- 更新 README.md 如果有新功能
- 添加代码注释
- 更新配置示例

感谢您的贡献！

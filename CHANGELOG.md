# 更新日志

所有项目的显著更改都将记录在此文件中。

## [1.1.0] - 2026-03-24

### Added
- 每3分钟自动检查一次服务状态
- 显示最后检查时间
- 实时服务状态指示器
- 自动日志刷新功能

### Improved
- 优化了 UI 界面设计
- 改进了日志显示（支持换行）
- 增强了报错弹窗展示
- 提升了服务稳定性
- 改为 GitHub 深色主题
- 优化了弹窗交互体验

### Fixed
- 修复日志加载问题
- 修复日志分行显示
- 优化报错处理机制
